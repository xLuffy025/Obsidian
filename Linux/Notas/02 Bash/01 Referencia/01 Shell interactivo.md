el Shell Bash se usa **comúnmente de forma interactiva:** te permite introducir y editar comandos, y luego ejecutarlos al teclear `Return`.
Muchos sistemas operativos basados y similares a Unix utilizan Bash como su shell predeterminado (especialmente Linux y macOS). El terminal introduce automáticamente un proceso interactivo de la shell de Bash en el inicio.

Salida `Hola Mundo` escribiendo lo siguiente:
```bash 
echo "Hola Mundo"
#> Hola Mundo # ejemblo de salida 
```

>[!nota] Notas
>- Puede cambiar el shell simplemente escribiendo el nombre del shell en la terminal. Por ejemplo `sh` `bash` `zsh` > 
>- `echo` Bash incorpora el comando que escribe los argumentos que reciben en la salida estándar. Por defecto, añade una nueva línea a la salida, por defecto.
