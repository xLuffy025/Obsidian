Una distribución de Linux (o "distro") ==es una versión funcional del sistema operativo Linux que agrupa el núcleo (kernel) de Linux con el software de sistema GNU, herramientas de gestión de paquetes (como APT o DNF), un entorno de escritorio (GNOME, KDE) y aplicaciones preinstaladas==. Se diseñan para satisfacer necesidades específicas, desde uso doméstico hasta servidores empresariales.

- **Componentes básicos:** Todas incluyen el mismo núcleo (kernel) Linux, pero se diferencian por la personalización, la gestión de software y el escritorio.

- **Gestión de software:** Utilizan repositorios y gestores de paquetes (ej. `.deb` para Debian/Ubuntu, `.rpm` para Red Hat/Fedora) que facilitan la instalación, actualización y eliminación de aplicaciones de forma segura.

- **Tipos de Distribuciones:**
   - **Comunitarias:** Gratuitas, mantenidas por la comunidad (ej. Debian, Arch Linux, Mint).

   - **Empresariales:** Con soporte técnico especializado (ej. Red Hat, SUSE).

Idea clave:
Las distros existen para distintos usos:
desktop, servidor, aprendizaje, empresa.
