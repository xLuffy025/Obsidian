## Idea clave:

Actualizar el sistema es obligatorio antes de instalar cualquier cosa.

**comandos**

Debian / Ubuntu
```bash 
sudo apt update && sudo apt upgrade -y
```

ArchLinux
```bash 
sudo pacman -Syu
```

Fedora 
```bash
dnf upgrade -y
```
