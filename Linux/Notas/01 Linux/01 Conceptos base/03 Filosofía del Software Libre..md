Qué entendí:

La filosofía del software libre se centra en la ética y la identidad del usuario garantizando cuatro fundamentos: 
ejecutar, estudiar, modificar y redistribuir el software, tal como promueve la Free Software Fundación. Basado en el proyecto GNU
y el núcleo Linux, busca la transparencia y colaboración, evitando el control propietario.

- **Las Cuatro Libertades Esenciales:
- **Libertad 0:** Ejecutar el programa para cualquier propósito.
- **Libertad 1:** Estudiar y modificar el código fuente.
- **Libertad 2:** Redistribuir copias.
- **Libertad 3:** Distribuir versiones modificadas.

- **Enfoque Ético:** La filosofía de Linux no se trata de precio(gratis), sino de control y libertad. Busca empoderar al usuario frente a las restricciones del software propietario.

- **Copyleft y GPL:** Muchas herramientas de Linux utilizan la Licencia Pública General (GPL) de GNU, que asegura que las libertades se mantengan en las versiones modificadas.

- **Filosofía Unix:** Fomenta un diseño de software modular, simple y claro 

Distinción mental:
- Software libre → ética y libertad
- Open Source → enfoque práctico y técnico

Por qué importa:
La distinción clave es que el software libre se enfoca en la ética, mientras que el "código abierto" (open source) suele centrarse más en la eficiencia técnica, aunque ambos comparten la apertura del código fuente.
